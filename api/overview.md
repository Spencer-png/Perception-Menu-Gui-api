# API Overview

- Frame: `BeginFrame()`, `EndFrame()`
- Windows: `BeginWindow()`, `EndWindow()`
- Child: `BeginChild()`, `EndChild()`
- Layout: `SameLine()`, `Indent()`, `Unindent()`, `Spacing()`, `Separator()`
- Text & Tips: `Text()`, `HelpMarker()`, `SetTooltip()`
- Widgets: `Button`, `Checkbox`, `RadioButton`, `SliderInt`, `SliderFloat`, `DragInt`, `DragFloat`, `InputText`, `InputTextEx`
- Selectors: `SingleSelect`, `MultiSelect`, `InputListSingle`, `InputListMulti`
- Color: `ColorPicker`, `CheckboxWithColorPicker`
- Keybind: `Keybind`
- Containers: `OpenPopup`, `CloseCurrentPopup`, `BeginPopup`, `EndPopup`; `BeginTabBar`, `BeginTabItem`, `EndTabItem`, `EndTabBar`; `BeginColumns`, `NextColumn`, `EndColumns`
- Side tabs: `SideTabBar`
- Demos: `ProDemoWindow`, `BasicDemoWindow`, `WidgetsDemoWindow`, `PopupsDemoWindow`, `StyleDemoWindow`
